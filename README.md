# spotify-api-page


Para instalar o projeto
```sh
 npm install
   ```
   ou
   ```sh
 yarn install
   ```

Para executar o projeto
 ```sh
 npm run start
   ```
 
 ou
 ```sh
 yarn run start
   ```
 

Para executar os testes
   ```sh
  npm run test
   ```
  
 Para testes com coverage
  ```sh
  npm run test:coverage
   ```
  
 Para rodar o build
  ```sh
  npm run build
   ```
  ou
   ```sh
 yarn run build
   ```
